### Instructions

Antes de executar o projeto, certifique-se de estar com a versão node 24.14.1 e depois execute o comando 
```
npm install
```

Utilize a versão Node 24.14.1


## Executando o projeto
Para executar o projeto no modo terminal execute o comando

```
npx cypress run
```

Para executar o projeto com interação via Browser execute o comando

```
npx cypress open
```
